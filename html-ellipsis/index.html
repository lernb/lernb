html溢出文字省略号处理

ler
2020-08-07  

  • white-space

    设置或检索对象内的文本显示方式

    white-space:normal;/* 默认处理方式 */
white-space:nowrap;/* 强制文本在同一行内显示，直到文本结束或者遇到br标签换行 */

  • text-overflow 文字溢出

    设置或检索是否使用一个省略标记（…）标示文本的溢出

    text-overflow:clip; /* 不显示省略标记（...）直接裁切 */
text-overflow:ellipsis; /* 文本溢出时显示省略标记（...） */

    注意：使用文本溢出省略标记（…）时，一定要先设置文本在同一行内显示

使用方式

/* 1. 首先强制同一行内显示文本 */
white-space: nowrap;
/* 2. 超出部分隐藏 */
overflow: hidden;
/* 3. 使用省略标记 */
text-overflow: ellipsis;
