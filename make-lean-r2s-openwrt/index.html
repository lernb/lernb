R2S Lean OpenWrt固件编译

ler
2020-10-09   转恩山无线论坛

Lean项目地址：https://github.com/coolsnowwolf/lede

一、安装Ubuntu 18 x64系统

安装系统和用SSH工具登录应该都会吧，此步省略！

更新系统：

sudo apt-get update&& sudo apt-get upgrade -y && sudo apt-get dist-upgrade -y && sudo apt-get clean && uname -a

二、搭建编译环境

2.1 搭建Lean源码编译环境

安装必需软件：

cd && sudo apt-get update[backcolor=rgba(27, 31, 35, 0.05)] && sudo apt-get -y install build-essential asciidoc binutils bzip2 gawk gettext git libncurses5-dev libz-dev patch python3.5 unzip zlib1g-dev lib32gcc1 libc6-dev-i386 subversion flex uglifyjs git-core gcc-multilib p7zip p7zip-full msmtp libssl-dev texinfo libglib2.0-dev xmlto qemu-utils upx libelf-dev autoconf automake libtool autopoint device-tree-compiler g++-multilib

下载源码：

cd && git clone https://github.com/coolsnowwolf/lede

更新安装源码：

cd ~/lede/ && git pull && ./scripts/feedsupdate -a && ./scripts/feedsinstall -a

2.2 搭建FriendlyWRT编译环境

官方地址：http://wiki.friendlyarm.com/wiki/index.php/How_to_Build_FriendlyWrt/zh

安装必需软件：

cd && sudo wget -O - https://raw.githubusercontent.com/friendlyarm/build-env-on-ubuntu-bionic/master/install.sh | bash

安装repo工具：

cd && git clone https://github.com/friendlyarm/repo && sudo cp repo/repo /usr/bin/

下载源代码：

  • 方式一：从github下载完整的源代码

    cd && sudo mkdir friendlywrt-rk3328 && cd ~/friendlywrt-rk3328 && sudo repo init -u https://github.com/friendlyarm/friendlywrt_manifests -b master-v19.07.1 -m rk3328.xml --repo-url=https://github.com/friendlyarm/repo --no-clone-bundle && sudo repo sync -c --no-clone-bundle

  • 方式二：网盘下载(以NanoPi R2S为例)
    参考地址：http://wiki.friendlyarm.com/wiki/index.php/NanoPi_R2S/zh
    网盘链接：http://download.friendlyarm.com/nanopir2s
    文件位于以下路径：sources/friendlywrt-rk3328-20210326.tar

文件下载后上传至编译机/home/用户名/路径下，然后用一下命令解压更新：

cd && 7z x friendlywrt-rk3328-20200226.7z && cd ~/friendlywrt-rk3328 && repo sync -l --no-clone-bundle && repo sync --no-clone-bundle

如果linux-libc-dev:i386无法安装，使用以下命令：

sudo dpkg --add-architecture i386 && sudo apt update && sudo apt -y install linux-libc-dev:i386

三、调整文件

3.1 基于FriendlyWRT导出Lean相关文件

导出feeds.conf.default文件：

cp -rf ~/lede/feeds.conf.default ~/friendlywrt-rk3328/friendlywrt/

导出luci文件：

cp -rf ~/lede/package/lean ~/friendlywrt-rk3328/friendlywrt/package

3.2 基于FriendlyWRT添加Lean相关依赖（以ucl和upx为例）

导出文件：

cp -rf ~/lede/tools/ucl ~/friendlywrt-rk3328/friendlywrt/tools/ && cp -rf ~/lede/tools/upx ~/friendlywrt-rk3328/friendlywrt/tools/

编辑文件：

vi ~/friendlywrt-rk3328/friendlywrt/tools/Makefile

将tools-y += ucl upx添加到# subdirectories to descend into段里，例：
tools-$(CONFIG_TARGET_tegra) += cbootimage cbootimage-configs的下一行添加：tools-y += ucl upx。

将$(curdir)/upx/compile := $(curdir)/ucl/compile添加到# builddir dependencies段里，例：
$(curdir)/cbootimage/compile += $(curdir)/automake/compile的下一行添加$(curdir)/upx/compile := $(curdir)/ucl/compile。

四、更新源码及固件配置

更新源码：

cd ~/friendlywrt-rk3328/friendlywrt && ./scripts/feeds update -a && ./scripts/feeds install -a

设置固件访问IP：

sed -i 's/192.168.2.1/X.X.X.X/g' ~/friendlywrt-rk3328/friendlywrt/package/base-files/files/bin/config_generate

XXXX部分更改成自己想要的网关IP

4.3 配置固件：

cd ~/friendlywrt-rk3328/friendlywrt && make menuconfig

基本选项设置（每项前部有*标志表示已选中，使用空格键勾选）

  • Target System选择rockchip rk3328 (aarch64)
  • Target Images选择squashfs
  • Base system取消dnsmasq选择 dnsmasq-full
  • Base system选择dnsmasq-full
  • LuCI>hemes选择luci-theme-bootstrap
  • LuCI>Applications选择自己想要的
  • Network>File Transfer选择wget
  • Network>IP Addresses and Names选择 ddns-scripts
  • Network>IP Addresses and Names选择 ddns-scripts_aliyun
  • Network>IP Addresses and Names选择 dns-scripts_no-ip_com

五、编译

以防万一被退出导致编译终止，使用screen命令创建编译：

  • 创建运行窗口：screen -S r2s，r2s为自定义的窗口名称；
  • 查看已有的screen窗口：screen -ls；
  • 返回已有的screen窗口：screen -d -r r2s，r2s为窗口名称。

编译命令：

cd ~/friendlywrt-rk3328/friendlywrt && make -j线程数 download v=s && cd ~/friendlywrt-rk3328 && ./build.sh nanopi_r2s.mk

线程数根据本机环境设置。

等待编译完成，固件生成目录为~/friendlywrt-rk3328/scripts/sd-fuse/out。
